# Pentaroot
Pentaroot adalah program sederhana, dimana program ini dibuat semata-mata hanya untuk mempermudah anda dalam proses penginstalan kebutuhan-kebutuhan yang ada di Linux.
## Contohnya 
- [x] Fcrackzip
- [x] GoldenEye
- [x] Ip Tracer
- [x] Nmap
- [x] Oh My Zsh
- [x] Pwnloris
- [x] Python3
- [x] Redhawk
- [x] Rockyou
- [x] Xerxes
